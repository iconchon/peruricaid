Peruri Code

Kode unik yang diproduksi sistem Peruri pada label sekuriti dokumen dan atau media lainnya untuk menyimpan informasi public & private di dalamnya. Peruri Code memiliki peranan penting untuk Penjaminan keaslian (authentication), kontrol distribusi (supply chain) dan penjaga brand image (brand protection) guna mempertahankan dan meningkatkan kepuasan pelanggan terhadap suatu produk komersil.

Peruri QR

Merupakan Code yang berisi informasi suatu produk sebagai salah satu fitur untuk melakukan pengamanan keaslian produk atau barang. Fitur sekuriti menyimpan informasi public & private sehingga dapat memberikan jaminan bahwa dokumen dienkripsi dan aman.

Peruri Seal

kode yang menyimpan informasi seperti bingkai foto yang bisa digunakan sebagai Stempel Digital suatu organisasi. Peruri Seal tidak dapat dibaca menggunakan aplikasi scanner biasa, melainkan menggunakan scanner khusus yang sudah tertanam dalam sebuah SDK milik Peruri yang bisa diimplementasikan ke mobile application.

Keyla (Key for Digital Authentication)

Peruri Token adalah multifactor authentication yang merupakan dynamic key sebagai One Time ID sebagai kode verifikasi user yang dapat digunakan pada berbagai use case, sebagai berikut:

Keyla For:
  • OTP Code
  • Soft Token untuk Internet Banking
  • Login Code
